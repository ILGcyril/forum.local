#Форум на PHP

Простой форум на PHP с использованием MySQL и PDO. Проект предназначен для обучения и 
демонстрации базовых возможностей веб-приложений: регистрация, авторизация, создание и редактирование постов.



Возможности

- Регистрация и авторизация пользователей
- Создание, редактирование и удаление постов
- Просмотр всех постов в ленте
- Просмотр постов конкретного пользователя
- Простая система сессий для авторизации



Технологии

- PHP 8+
- MySQL / MariaDB
- PDO для безопасного взаимодействия с базой данных
- HTML для фронтенда (без CSS и JS)



Установка

1. Склонируйте репозиторий или скачайте проект.

2. Выполните SQL-скрипт для создания таблиц:
    ```bash
    mysql -u [username] -p < setup.sql

3. Настройте соединение с базой данных в config/config.php:
    define('HOST', 'localhost');
    define('USER', 'ваш_пользователь');
    define('PASSWORD', 'ваш_пароль');
    define('DBNAME', 'forum');

4. Разместите проект в папке веб-сервера (например, htdocs или www).

5. Откройте в браузере главную страницу:
    http://localhost/forum/index.php



Примечания

- Пароли пользователей хранятся в базе данных в захешированном виде (password_hash()).
- Для безопасности рекомендуется запускать проект в локальной среде с включённой поддержкой HTTPS.
- Скрипт setup.sql создаёт базовые таблицы: users и posts.



Проект создан как учебный пример простого форума на PHP с MySQL.
